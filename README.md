**【群聊行为守则】**  

**第一章 总则**  
1. 本群为 ______ （群名称） 交流平台，旨在 ______ （群定位，如学习、工作、兴趣等），请所有成员本着友好、尊重、文明的原则参与讨论。  
2. 加入本群即视为已阅读并认同本守则，遵守本守则的全部内容。  

**第二章 成员基本义务**  
1. **实名制**：成员应使用真实的微信/QQ/钉钉等帐号昵称，禁止冒用他人身份。  
2. **信息真实性**：发布的文字、图片、音视频等信息必须真实可靠，严禁散布谣言、虚假信息。  
3. **尊重他人**：禁止人身攻击、歧视、辱骂、挑衅等言行，维护和谐交流氛围。  
4. **保密义务**：未经授权不得泄露群内涉及的商业机密、个人隐私或其他受法律保护的敏感信息。  

**第三章 禁止发布的内容**（除本守则另有规定外，凡违者均视为违规）  
1. **政治类**  
   - 反对宪法所确定的基本原则、否定中国共产党领导、否认社会主义制度的言论。  
   - 煽动、宣传颠覆国家政权、推翻社会主义制度的内容。  
2. **法律法规禁止的内容**  
   - 违反《中华人民共和国刑法》第105条、106条等规定的颠覆、分裂、煽动等信息。  
   - 传播暴力恐怖、极端主义、犯罪教唆、非法武器、毒品、赌博、淫秽色情等。  
   - 侵犯他人著作权、商标权、专利权等知识产权的内容。  
3. **商业广告类**  
   - 未经管理员授权的商业推广、传销、直销、招聘广告、软文植入。  
4. **其他不当内容**  
   - 侮辱、诽谤、攻击他人隐私的言论；  
   - 含有恶意软件、病毒、钓鱼链接的任何文件或链接。  

**第四章 违规处理流程**  
| 违规次数/情节 | 处理方式 | 备注 |
|---|---|---|
| 第一次轻微违规（如违规言论、广告） | 删除违规信息，警告一次 | 记录违规时间、内容、成员ID |
| 第二次违规（24 小时内） | 删除并禁言 7 天 | 保留截图、聊天记录 |
| 第三次违规或情节严重（涉及违法、散布谣言、暴力等） | 移出群聊，永久封禁；必要时上报平台或主管部门 | 提交全部证据材料，配合调查 |
| 触犯刑法或行政法规 | 立即移除并向公安、网信部门报告 | 按《刑法》第105条等依法追究责任 |

**第五章 管理员职权与义务**  
1. 管理员有权对违规信息进行删除、禁言、移除成员的操作。  
2. 管理员需及时记录处理过程，确保公平公正。  
3. 如遇重大争议，管理员应及时将问题上报群主或相关监管部门。  

**第六章 申诉与救济**  
1. 成员对禁言、移除等处理有异议，可在 **24 小时** 内向管理员提交书面申诉。  
2. 管理员须在收到申诉后 **12 小时** 内核查并反馈结果。若确认处理不当，可撤销处罚并恢复相应权限。  

**第七章 附则**  
1. 本守则自发布之日起生效，凡本群成员均应遵守。  
2. 本守则的修改、解释权归 **群主/管理员** 所有，必要时将以公告形式提前告知全体成员。  
3. 如本守则与国家法律、行政法规相冲突，以国家法律为准。  

---

> **温馨提示**：和谐的交流环境离不开每位成员的自觉维护，请大家共同遵守本守则，让本群成为一个安全、友好、富有建设性的交流平台。祝大家聊天愉快！
